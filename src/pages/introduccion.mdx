# Introducción a Scrypto

<Notification type="info">
  <Icon pack="ion" name="information-circle-outline" />
  <p>
    <strong>Info:</strong> Este manual de iniciación esta basado en la <a href="https://github.com/radixdlt/radixdlt-scrypto/releases/tag/v1.0.0">versión 1.0.0 de Scrypto</a>.
  </p>
</Notification>

Scrypto es un lenguaje específico de dominio [(DSL)](https://es.wikipedia.org/wiki/Lenguaje_espec%C3%ADfico_de_dominio) creado por [RadixDLT](https://radixdlt.com/) 
para permitir el desarrollo rápido, intuitivo y seguro de aplicaciones descentralizadas 
[(dApps)](https://es.wikipedia.org/wiki/Aplicaci%C3%B3n_descentralizada). 
Las dApps en Radix se forman a partir de 'componentes', también conocidos como [contratos inteligentes](https://es.wikipedia.org/wiki/Contrato_inteligente), 
que se ejecutan en el ['Radix Engine v2'](https://www.radixdlt.com/blog/radix-engine-v2-an-asset-oriented-smart-contract-environment) . 
Scrypto toma prestada la sintaxis del lenguaje [Rust](https://www.rust-lang.org/), aprobechando al máximo su compilador, heredando su integridad de Turing e incluyendo características específicas [orientadas a activos](https://www.radixdlt.com/blog/scrypto-an-asset-oriented-smart-contract-language) para el desarrollo de aplicaciones seguras basadas en un paradigma de FSM [(máquina de estado finitos)](https://www.radixdlt.com/blog/is-scrypto-turing-complete).

### Instalación de herramientas y simulador

<Notification type="info">
  <Icon pack="ion" name="information-circle-outline" />
  <p>
    <strong>Info:</strong> Las siguientes intrucciones están basadas en la <a href="https://docs.radixdlt.com/docs/getting-rust-scrypto">documentación oficial</a>.
  </p>
</Notification>

<div class="space-content">
  <h4>Instalación Rust y herramientas</h4>
  Para comenzar a trabajar con Scrypto, primero debe preparar tu sistema para el desarrollo de Rust.
  <Accordion>
    <AccordionItem header="Windows">
      **Primero:** asegurate de que tienes instalado [*git*](https://git-scm.com/), puedes ejecutar la siguiente instrucción:
      ```powershell
      git -v
      ```
      el resultado deberia ser algo parecido a esto: 
      ```powershell
      git version 2.39
      ```
      si no entonces puedes instalarlo desde el siguiente enlace -> [*Instalar Git*](https://git-scm.com/downloads)

      **Segundo:** habilita la compatibilidad con rutas largas de git con la siguiente instrucción:
      ```powershell
      git config --system core.longpaths true
      ```

      **Tercero:** Instalaremos "Desktop development with C++" que se encuentra dentro de [Build Tools for Visual Studio 2019](https://visualstudio.microsoft.com/thank-you-downloading-visual-studio/?sku=BuildTools&rel=16)

      **Cuarto:** Descargaremos e instalaremos [rustup-init.exe](https://win.rustup.rs/x86_64)

      **Quinto:** Instalar [LLVM 13.0.1](https://github.com/llvm/llvm-project/releases/download/llvmorg-13.0.1/LLVM-13.0.1-win64.exe) Importante: Asegurate de marcar la opción que agrega LLVM a la ruta del sistema.

      **Sexto:** Habilitamos *cargo* desde la linea de comando (PowerShell):
      ```powershell
      set path=%USERPROFILE%/.cargo/env
      ```

      **Septimo:** Añadimos WebAssembly target desde la misma linea de comando:
      ```powershell
      rustup target add wasm32-unknown-unknown
      ```
    </AccordionItem>
  </Accordion>
</div>

Finalmente instala el Simulador de Radix Engine y la línea de comandos con las siguientes instrucciones (el proceso puede durar unos minutos): 
```powershell
git clone https://github.com/radixdlt/radixdlt-scrypto.git
cd radixdlt-scrypto
cargo install --path ./simulator
```
